# Real Estate Digital Twin Website

This repository contains a modern website that explores the concept of digital twins in real estate. It is built using HTML, CSS, JavaScript, and incorporates animations and interactive features with GSAP and Locomotive Scroll.

## Project Overview

The project showcases the following features:

- **Navigation Bar:** Contains logo and buttons for booking a demo and opening a menu.
- **Hero Section:** Includes a video background and call-to-action buttons.
- **Multi-Page Structure:** The website consists of several pages that discuss the concept of digital twin tokens, how they connect stakeholders, and their benefits.
- **Interactive Animations:** Utilizes GSAP and Locomotive Scroll to create smooth scrolling animations and other visual effects.
- **Canvas Animations:** Certain sections use canvas animations to enhance the user experience.

## Code Structure

- **HTML:** Provides the structure of the website, defining the different sections and elements.
- **CSS:** Styles the website, ensuring a consistent layout and responsive design across various screen sizes.
- **JavaScript:** Includes scripts to handle animations, scroll events, and other interactive features.

## External Libraries

- **Remixicon:** Used for icons in the website.
- **Locomotive Scroll:** Enables smooth scrolling and scroll-based animations.
- **GSAP (GreenSock Animation Platform):** Used for advanced animations and scroll-based triggers.

## Usage

To view and interact with the website:

1. Clone or download the repository to your local environment.
2. Open the `index.html` file in a web browser.
3. Explore the website to see the various animations, transitions, and interactive elements.
4. Customize the content and styles to fit your own project needs.

## Contributing

Contributions are welcome! If you'd like to suggest improvements or add new features, please open an issue or submit a pull request.
